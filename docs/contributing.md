# Contribuciones

Sigue estas pautas para colaborar en el proyecto:

1. Haz un fork del repositorio.
2. Crea una rama descriptiva: `feature/<tarea>` o `bugfix/<problema>`.
3. Realiza commits pequeños y significativos.
4. Abre un pull request cuando tu tarea esté lista.
5. Revisa y resuelve comentarios antes de fusionar.

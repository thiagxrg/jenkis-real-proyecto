# Arquitectura del Proyecto

Este proyecto está dividido en dos capas principales:

- Frontend: Interfaz web para usuarios, desarrollada con componentes reutilizables.
- Backend: API REST que gestiona eventos, usuarios e inscripciones.

## Frontend

- `public/`: Recursos estáticos.
- `src/components/`: Componentes UI.
- `src/pages/`: Páginas principales.
- `src/services/`: Conexión con la API.
- `src/styles/`: Estilos globales y componentes.

## Backend

- `src/controllers/`: Controladores HTTP.
- `src/models/`: Modelos de datos.
- `src/routes/`: Rutas y endpoints.
- `src/services/`: Lógica de negocio.
